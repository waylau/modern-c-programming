# Modern C Programming. 现代C语言编程实战

## 内容简介


现代C语言编程实战，基于最新C18的标准。学习C语言的同志们，燥起来！

技术版本如下：

* MinGW-w64 v9.0.0
* GNU C Compiler v8.1.0
* Visual Studio v17.0.4
* ...
